# Support

Last updated: 2026-03-14

## Get help

- Open an issue: https://github.com/dnbc-edu/qti-word-addin/issues
- Discussions (if enabled): https://github.com/dnbc-edu/qti-word-addin/discussions

## What to include in a report

- Word platform and version (Windows, macOS, or Web).
- Steps to reproduce.
- Input sample (if shareable) and generated ZIP details.
- Any console/error output.

## Response scope

Support is best-effort for this open-source project.

## Security issues

Please do not post sensitive data publicly. For potentially sensitive reports, open a private contact channel before sharing details.
